HTML Paragraph Element

Introduction

The <p> element is a very common element that contains text.

Usage

The <p> element is used where the user wants to add longer text paragraphs to a website, as <p> automatically inserts linebreaks which helps with formatting for different screen sizes.

Example

Here's an example of the <p> element:

                
<p>This text will be automatically formatted, 
for example extra spaces     will    be   removed.</p>
<p>The text can even have 
line breaks in the code and nothing will change on the website, 
beacuse of this paragraphs are very convenient and 
lines dont have to be very long in the code</p>
