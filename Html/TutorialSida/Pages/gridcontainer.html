Understanding CSS Grid Containers

In CSS Grid, the grid container is the parent element that holds and controls the layout of its child elements, known as grid items.

Key features of grid containers:

  • display: The "grid" value is applied to the container to enable the grid layout.
  • grid-template-columns: Defines the number and size of columns in the grid.
  • grid-template-rows: Specifies the number and size of rows in the grid.
  • grid-template-areas: Allows you to name grid areas and assign items to those areas.
  • grid-template: A shorthand property to set both columns and rows in one declaration.
  • grid-column-gap/grid-row-gap: Sets the size of the gap between columns/rows.
  • grid-gap: A shorthand for setting both column and row gaps.
  • justify-items/align-items: Aligns items along the row/column axis.
  • justify-content/align-content: Aligns the grid along the row/column axis when there's extra space.
