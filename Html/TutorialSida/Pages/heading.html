HTML Header Element

Introduction

The heading elements are a group of elements, the header elements are <h1>, <h2> and so on up to <h6>.

Usage

The heading elements are commonly used to start a paragraph and the <h1> is important for SEO as it makes googles crawler more likely to understand the contents of your website.

Example

Here's an example using some of the heading elements:

<h1>Welcome to My Website</h1>
<h2>Welcome to My Website subheading</h2>
<p>Text text text</p>
